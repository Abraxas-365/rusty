## Getting Started

1. Clone this repo.

2. Create an environment variable `OPENAI_KEY` with your API key from [OpenAI](https://openai.com/api/).

3. Build the Rust program.

```bash
cargo build --release
```

4. Add the binary to your path. Under target/release.
